**"Your Favorite DeskBuddy"**

FRIDAY is a futuristic, highly polished, Iron-Man-inspired desk companion that combines productivity, utilities, engineering tools, weather information, time management, and entertainment into a single handheld device built on the ESP32.

## 🌟 Features (Version 1.0)
- **Radial Arc Reactor Menu**: Smooth, rotating, futuristic circular UI.
- **Clock & Timers**: Digital/Analog clock, stopwatch, and countdown timer.
- **Weather Station**: Real-time temperature, humidity, and weather conditions.
- **Smart Calculator**: Basic arithmetic to advanced trigonometry.
- **System Monitor**: View free RAM, CPU freq, WiFi status, and uptime.
- **Engineering Toolbox**: Ohm's Law, Voltage Divider, RC Time Constant, etc.
- **Pomodoro Timer**: Boost productivity with a built-in focus timer.
- **DeskBuddy Mode**: Idle mode with motivational quotes, engineering jokes, and AI-like status updates.
